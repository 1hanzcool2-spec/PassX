# PassX Password Manager App

A desktop application version of the PassX Password Manager.

## Features

- Add username and password entries
- Display entries in a list
- Download passwords as a formatted text file
- Download app button (placeholder)

## Installation

1. Ensure Python is installed on your system.
2. Install required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

Run the application:
```
python main.py
```

## Note

Tkinter is included with Python, so no additional installation is needed for the GUI.
